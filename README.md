# Ecommerce Data Platform

This project simulates a real-world data platform for an e-commerce company starting its digital operations.

## Project Goals
- Analyze sales and customer data
- Compare product prices with the market
- Generate data-driven insights
- Support decision-making using data

## Tech Stack
- Python (Pandas)
- PostgreSQL (Supabase)
- dbt
